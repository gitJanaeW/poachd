# poachd

## Purpose

Find new recipes, modify them to your tastes and share your "poachd" version of the recipe with the world! Paid subscriptions to the website will earn you access to unlimited poaches every month.

## Built With

- MongoDB
- Express
- React
- Node
- Stripe
- Redis
- Webpack
